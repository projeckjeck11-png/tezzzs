# MetricDeck

MetricDeck is a professional tool for setting and inputting time data, analyzing timeline graphs, and calculating productivity and efficiency.

Created by: (linkedin)

## Features
- Structured time input and timeline setup
- Timeline graph analysis and visualization
- Productivity and efficiency calculation workflow

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

### Prerequisites
- Node.js (LTS recommended)
- npm

### Install
```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
```

### Development
```sh
npm run dev
```

### Build
```sh
npm run build
```

### Preview
```sh
npm run preview
```

## Notes for Developers
- Keep commits small and focused.
- Prefer running `npm run dev` while iterating on UI changes.
- Validate key workflows after changes to timeline logic.
